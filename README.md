# Portafolio Futurista con Hero Animado y Partículas

Este proyecto es un portafolio web profesional construido con **React + Vite**, **TailwindCSS**, **Framer Motion** y un sistema de partículas hecho con **JS + CSS puro**.  
El objetivo es mostrar tu perfil como desarrollador full stack con un estilo **futurista, premium y enfocado en ciberseguridad**.

---

## Características principales
- **Hero animado** con avatar, nombre y título usando Framer Motion.
- **Fondo oscuro dinámico** (`bg-black`) con partículas translúcidas flotando como esporas.
- **Partículas personalizadas** hechas con JavaScript y CSS, sin librerías externas.
- **Diseño responsive** gracias a TailwindCSS.
- **Animaciones premium** con sombras, glow y efectos difusos.
